# Savdo-(E) Web Frontend

This is the React frontend application for the Savdo-(E) e-commerce platform.

## Features
- **React 18**: Modern UI library for fast and interactive interfaces.
- **Vite**: Rapid development environment and production build tool.
- **Component-Based Architecture**: Reusable UI components.
- **State Management**: Redux/Zustand for global application state.
- **Tailwind CSS**: Utility-first styling for premium design.

## Folder Structure
- **assets**: Static resources like images, fonts, and global styles.
- **components**: Reusable UI components (buttons, cards, headers).
- **pages**: Main views for different routes (home, product, cart, login).
- **layouts**: Structural components wrapping various pages.
- **hooks**: Custom React hooks for shared logic.
- **services**: API interaction and external data fetching.
- **store**: Global state management configuration.

## Installation
1. Navigate to the web directory:
   ```bash
   cd web
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

- **Aesthetic Excellence**: Premium minimalist design with modern animations.
- **Responsive Layout**: Optimized for desktop, tablet, and mobile browsers.
