# Savdo Admin Panel

Savdo platformasini boshqarish uchun React + Tailwind CSS asosida qurilgan admin interfeys.

## Tezkor ishga tushirish

```bash
npm install
npm run dev    # http://localhost:5174
```

Demo kirish: `admin@savdo.uz` / (istalgan parol)
