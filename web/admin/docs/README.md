## Loyiha haqida qisqacha

**Savdo Admin Panel** — bu savdo platformasini boshqarish uchun yaratilgan React + Tailwind CSS asosidagi admin interfeys.

### Texnologiyalar

- **Framework:** React 18 + Vite
- **Stil:** Tailwind CSS
- **Routing:** React Router DOM v6
- **HTTP:** Axios
- **I18n:** Uzbek / English / Russian
- **Tema:** Dark mode (default)

### Tezkor ishga tushirish

```bash
npm install
npm run dev    # http://localhost:5174
```
